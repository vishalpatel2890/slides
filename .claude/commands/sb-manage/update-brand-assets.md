# Slide Builder - Update Brand Assets Command

This command manages the brand asset catalog for icons, logos, and images used in slides.

<steps CRITICAL="TRUE">
1. Read the workflow configuration at @.slide-builder/workflows/update-brand-assets/workflow.yaml
2. Read the instructions at @.slide-builder/workflows/update-brand-assets/instructions.md
3. Execute the workflow steps in order, engaging in conversation with the user
4. Scan existing assets or import new ones based on user selection
5. Save catalog files with semantic tags for each asset
</steps>
