# Use the viewer

Navigate the interactive presentation viewer.

## Opening the viewer

After building slides:

```bash
open output/{deck-slug}/index.html
```

Works with `file://` protocol (no server required).

## Gallery view

The default view shows:
- Deck title
- Slide count
- Thumbnail grid

Click any thumbnail to open that slide in presentation mode.

## Presentation mode

Full-screen slide display with:
- 1920x1080 viewport
- Navigation controls
- Slide counter

### Navigation

| Action | Method |
|--------|--------|
| Next slide | Click right arrow, press Right/Space |
| Previous slide | Click left arrow, press Left |
| Jump to slide | Click thumbnail in strip |
| Exit to gallery | Press Escape |

## Editing text

Text elements are editable:
1. Click any text in presentation mode
2. Edit content directly
3. Changes save automatically to slide state

## Slide thumbnails

The thumbnail strip at the bottom shows:
- Current slide indicator
- Preview of all slides
- Click to jump

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| Left/Right | Navigate slides |
| Space | Next slide |
| Escape | Exit presentation |
