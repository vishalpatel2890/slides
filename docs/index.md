## What is Slide Builder?

Slide Builder is a CLI-based framework that enables you to create professional, brand-consistent presentations using natural language. It integrates with Claude Code to provide an agentic workflow for:

- Extracting brand identity from websites, PDFs, and images
- Generating complete theme systems with colors, typography, and layouts
- Planning and building presentation decks with AI assistance
- Editing slides using natural language commands
- Exporting to Google Slides

## Key features

- **Brand theme extraction**: Analyze websites and assets to create consistent themes
- **Template catalog**: 6+ built-in templates with intelligent matching
- **Natural language editing**: "Make the title bigger" style commands
- **Text preservation**: User edits survive layout regenerations
- **Interactive viewer**: Gallery and fullscreen presentation modes
- **Google Slides export**: Direct integration via OAuth

## Requirements

- Claude Code CLI
- Node.js (for scripts)
- Modern browser (for viewing presentations)
