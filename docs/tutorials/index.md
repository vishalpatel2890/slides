## Prerequisites

Before starting these tutorials, ensure you have:
- Claude Code CLI installed
- Basic familiarity with command-line interfaces
- A project folder for your presentations

## Learning path

We recommend following tutorials in this order:

1. Start with [Your first presentation](first-presentation.md) to understand the basic workflow
2. Continue to [Brand theme setup](brand-theme-setup.md) for customization
3. Explore [Full deck workflow](full-deck-workflow.md) for comprehensive projects
4. Learn [Template creation](template-creation.md) when you need specialized layouts

After completing tutorials, see [How-to guides](../how-to/index.md) for specific tasks.
