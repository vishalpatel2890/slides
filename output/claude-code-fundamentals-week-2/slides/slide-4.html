What Happens at the Limit

Context Limits & Auto-Summarization

Claude Code
Stage 1

Normal Operation

Full conversation history maintained. All messages, code, and context available.

Context usage: 0-80%
Stage 2

Approaching Limit

Context getting full. Claude Code detects this and prepares for summarization.

Context usage: 80-95%
Stage 3

Auto-Summarization

Older conversation compressed into summary. Key decisions and context preserved.

Frees ~50% of context
Stage 4

Continued Work

Session continues with summarized history. Recent context remains detailed.

Ready for more work!

Key insight: Auto-summarization preserves important decisions and context, but some detail is lost. For complex tasks, consider using /compact proactively or starting fresh sessions.
