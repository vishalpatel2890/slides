Session Management

Session Commands: /resume & /compact

Claude Code
/resume

Continue previous session with full context preserved

Terminal
$ claude --resume
Resuming session from 2 hours ago...
Context: 47,832 tokens loaded
✓ Session resumed successfully
 
Last message: "Now let's add tests..."
 
Ready to continue where you left off
Best For

Coming back to a complex task after a break, preserving all decisions and context from before.

/compact

Compress context to free up space for new work

Terminal
> /compact
Analyzing conversation history...
Before: 156,234 tokens (78%)
After: 52,108 tokens (26%)
 
✓ Freed 104,126 tokens (52%)
 
Key decisions and context preserved
Best For

Long sessions approaching context limits. Proactively run this before hitting automatic summarization.
