What is a Context Window?

Think of it as Claude's "working memory" — everything Claude can "see" at once to understand your request and generate a response.

Capacity ~200K tokens

Claude Code has a massive context window — enough for substantial codebases and long conversations.

Tokens vs Words

A token is roughly 4 characters or ¾ of a word. Code tends to use more tokens due to syntax characters and formatting.
