Quick Reference

Essential Commands

Claude Code
/help

Display available commands and their descriptions. Your starting point when lost.

When to use
First time using Claude Code or need command reference
/clear

Clear the conversation history. Fresh start without quitting the session.

When to use
Context is polluted or switching to unrelated task
/compact

Compress context to free up space. Preserves key decisions and summaries.

When to use
Long session approaching context limits
/resume

Continue from a previous session with full context restored.

When to use
Coming back to continue previous work
/status

Check current session status including context usage and model info.

When to use
Monitor context usage during long sessions
/config

View or modify Claude Code configuration settings.

When to use
Adjust settings like model, permissions, or defaults
