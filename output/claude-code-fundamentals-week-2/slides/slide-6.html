Decision Framework

Fresh Start vs Resume

Claude Code

Fresh Start

When to use
  • Starting a completely new task or project
  • Context is polluted with irrelevant info
  • Claude keeps referencing old, wrong decisions
  • Switching to a different codebase
claude New session

Resume

When to use
  • Continuing work from previous session
  • Need to preserve decisions made earlier
  • Iterating on a complex implementation
  • Coming back after a break
claude --resume Resume session
