CLI Navigation Basics

Essential commands to navigate and control your Claude Code sessions.

  • claude Start an interactive session in the current directory
  • /help Display all available commands and keyboard shortcuts
  • /clear Reset conversation context and start fresh
  • /compact Summarize conversation to reduce context length
  • /cost Show current session token usage and cost
  • Ctrl+C Cancel current operation or exit the session
