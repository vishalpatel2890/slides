What Claude Code Can Do

A comprehensive set of tools that let you work alongside an intelligent agent in your development environment.

  • Read Your Entire Codebase

    Understands context across multiple files and can navigate complex project structures

  • Write and Edit Code

    Creates new files, modifies existing code, and applies changes across multiple locations

  • Run Shell Commands

    Executes terminal commands, runs tests, and sees the results to iterate on solutions

  • Iterate Based on Feedback

    Learns from errors, adjusts approach, and continuously improves until the task is complete

  • Use Tools: Search, Web, Files

    Searches your codebase, fetches web content, and manages files as part of complex workflows
